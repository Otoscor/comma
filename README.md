# comma

> 웹소설을 AI로 시각화하는 소셜 독서 플랫폼

**프로젝트 상태**: Planning & Design Phase
**최종 업데이트**: 2026-02-04
**작성자**: Youngzoo

---

## 📖 프로젝트 소개

**comma**는 웹소설 독자를 위한 차세대 독서 플랫폼입니다.

텍스트로만 상상하던 소설 속 장면을 AI 기술로 **이미지, 비디오, 음성**으로 구현하고, 다른 독자들의 시각화를 **소셜 플랫폼**에서 공유·발견할 수 있습니다. 읽다가 마음에 드는 문장을 드래그하면 AI가 바로 시각화해주고, 다른 독자들은 어떤 장면을 어떻게 상상했는지 본문 위의 **마커**를 통해 자연스럽게 발견할 수 있습니다.

### 서비스명 의미
**comma (,)**: 문장의 쉼표처럼, 독서의 흐름 속에서 잠시 멈춰 상상을 현실로 만드는 순간

---

## 🎯 핵심 가치

### 문제
- 웹소설 독자들은 텍스트만으로 장면을 상상해야 하는 한계
- 작가의 묘사와 독자의 상상 사이의 간극
- 같은 소설을 읽는 독자들과 해석을 공유할 방법 부족

### 솔루션
✅ **텍스트 스크랩 → AI 멀티미디어 생성**
✅ **로그(Log): 모든 독자의 시각화가 모이는 소셜 공유 공간**
✅ **마커: 본문 위에 다른 독자들의 시각화 위치 표시**
✅ **같은 장면, 다양한 해석 발견**

### 핵심 개념 3가지

#### 1. 로그 (Log)
**해당 웹소설에 대한 모든 독자들의 시각화 콘텐츠가 모이는 공유 공간**
- 내가 시각화하면 자동으로 로그에 저장
- 다른 독자들의 시각화를 감상하고 반응 (좋아요, 댓글)
- 탭으로 필터링: 이미지 / 비디오 / 음성 / 메모

#### 2. 마커 (Marker)
**본문 텍스트 위에 표시되는 원형 인디케이터**
- 다른 독자가 해당 위치를 시각화했음을 표시
- 마커 클릭 → 해당 독자의 로그(콘텐츠) 감상
- 마커 밀도로 인기 장면 발견

#### 3. AI 생성 플로우
**텍스트 → 이미지/비디오/음성**
1. 독서 중 텍스트 드래그/선택
2. 플로팅 툴바에서 타입 선택
3. "만들기" → AI 생성 (로딩)
4. "로그에 넣기" → 자동 저장

## 🎨 기능 개요

### 웹소설 플랫폼 (기본)
- 웹소설 읽기 뷰어
- 읽기 진행률 트래킹
- 북마크 및 독서 기록

### AI 멀티미디어 생성 ⭐️
- **이미지 생성**: 장면, 캐릭터, 배경
- **비디오 생성**: 짧은 장면 영상화 (5~15초)
- **음성 생성**: 캐릭터 대사, 내레이션, 배경 사운드

### 로그 (소셜 플랫폼) ⭐️
- **로그 보기 토글**: 본문에 마커 표시/숨김
- **마커 클릭**: 다른 독자의 시각화 감상
- **로그 갤러리**: 작품별 모든 시각화 모아보기
- **소셜 기능**: 좋아요, 저장, 댓글

## 🧭 사용자 플로우

### A. 내가 시각화 만들기
```
웹소설 읽기
  → 텍스트 드래그/선택
  → 생성 타입 선택 (이미지/비디오/음성/메모)
  → "만들기" 클릭
  → AI 생성 (로딩)
  → 결과 확인
  → "로그에 넣기"
  → 로그에 자동 저장
```

### B. 다른 독자 시각화 감상하기
```
웹소설 읽기
  → "로그 보기" 클릭
  → 본문에 마커 표시됨
  → 마커 클릭
  → 해당 독자의 로그 감상
  → 좋아요, 저장, 댓글
```

### C. 로그 갤러리 탐색
```
하단 "로그" 탭 클릭
  → 해당 소설의 모든 시각화 표시
  → 탭으로 필터링 (이미지/비디오/음성/메모)
  → 콘텐츠 클릭하여 감상
  → 좋아요, 저장, 공유
```

## ✨ 핵심 차별점

1. **독서 + AI 생성의 완벽한 통합**: 읽다가 바로 생성, 끊김 없는 UX
2. **맥락 기반 생성**: 단순 프롬프트가 아닌 소설 전체 맥락 활용
3. **독자 중심**: 작가가 아닌 독자가 원하는 장면을 선택
4. **멀티모달**: 이미지, 비디오, 음성을 모두 지원
5. **텍스트 위의 시각적 레이어**: 본문 위에 마커로 다른 독자들의 시각화 표시
6. **집단 재해석 플랫폼**: 같은 소설, 같은 장면을 여러 독자가 다르게 해석
7. **발견(Discovery)**: 인기 장면, 창의적인 해석을 자연스럽게 발견
